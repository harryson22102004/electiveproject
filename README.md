This repository contains a full-stack Multi-Vendor E-commerce platform built for an academic project.

Structure
- `shopnexus-api/` — Spring Boot backend (APIs, JPA, security)
- `shopnexus-ui/` — React frontend (Vite, components, routes)
- `docs/` — Project report, slides and additional deliverables

Quick start

1) Backend (from repository root):

```bash
cd "shopnexus-api"
mvn clean package
mvn spring-boot:run
```

2) Frontend:

```bash
cd "shopnexus-ui"
npm install
npm run dev
```
